# 🚗 AI-Based Autonomous Navigation System

## 📌 Overview
This project simulates an AI-based autonomous navigation system where a virtual agent navigates from a start point to a goal while avoiding obstacles using the A* path planning algorithm.

## 🎯 Features
- Path Planning using A* Algorithm
- Obstacle Avoidance
- Grid-based Simulation
- Visualization using Matplotlib

## 🛠 Tech Stack
- Python
- NumPy
- Matplotlib

## 🧠 How It Works
1. A grid environment is created
2. Obstacles are placed in the grid
3. A* algorithm finds the shortest path
4. The path is visualized graphically
